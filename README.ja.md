## 1. 概要

SGABM-MarketDynamics-Engineは、先進的なマルチエージェントシミュレーション（ABM）プラットフォームであり、「ダブルカーボン」目標の下、グリーン電力市場における政策介入（供給側の財政税制優遇措置および需要側のグリーン電力取引政策）、発電事業者の戦略、電力消費事業者の行動が長期的にどのように連動して進化するかをシミュレートすることを目的としています。

本エンジンは、データ依存型スクリプトから**依存関係ゼロ、パラメータ駆動型**のエンジンへの移行を実現しており、研究者はパラメータ調整を通じて直接的に政策のシナジー効果、マルチエージェント間の相互作用、市場均衡を探索することができます。

## 2. 基本原理

本エンジンのシミュレーションロジックは、2つの相互に支え合う数学的フレームワークに基づいて構築されています。「政策学習-市場ゲーム」という2層の動的相互作用を通じて、グリーン電力市場の複雑な進化過程と政策連携メカニズムを共同で模倣します。

### 1. **3段階スタッケルベルグ・シーケンシャルゲーム**：

   市場運営のミクロコアを構築するために、本エンジンは3段階スタッケルベルグ・シーケンシャルゲームを採用しています。このフレームワークでは、グリーン電力事業者が先導権を持つ「リーダー」として、電力消費事業者がリーダーのシグナルに基づいて反応する「フォロワー」として機能し、政府はゲームのルール（政策）を設定し、そこから学習する外部調整役として機能します。これは、現実の政策先行、発電事業者投資、電力消費事業者消費という意思決定の時系列と権力構造を正確に描写しています。

#### 第1段階（グリーン電力事業者の初期意思決定）：
   この段階では、市場リーダーであるグリーン電力事業者は、政府の当期の財政税制優遇政策（法人税減免、投資補助金など）の認識と、市場需要に関する将来予測に基づき、長期的な利益最大化を目的として、予備的なグリーン電力市場価格とそれに対応する設備投資判断を同時に策定します。この判断は、不確実性下での予測であり、市場の機先を制し、設備配置を確定させることを目的としています。

#### 第2段階（電力消費事業者の対応）：
   グリーン電力事業者からの予備的な価格シグナルを観測した後、フォロワーである電力消費事業者は意思決定を開始します。彼らは、外生的に与えられた電力総需要と政府による強制的なグリーン電力消費割当（RPS）という二重の制約下で、自らの購電総コストを最小化することを目的として、グリーン電力と従来型電力の最適な調達構成を計算します。このプロセスは本質的に制約付き線形最適化問題であり、その解はグリーン電力に対する実際の市場需要曲線を形成します。

#### 第3段階（グリーン電力の最終価格決定と市場クリアリング）：
   グリーン電力事業者は、第2段階から集約された実際の市場需要を観測し、価格調整メカニズムを起動します。彼らはベイズ更新の考え方に基づき、自らの生産コスト、初期価格提案、市場からフィードバックされた支払意思を総合的に考慮して、予備的な価格提案を修正し、最終的な公示価格を形成します。その後、取引所は全てのグリーン電力事業者からの最終供給曲線と電力消費事業者からの需要曲線を集約し、「価格優先、需給一致」の原則に従って統一クリアリングを実施し、最終的なグリーン電力取引量、クリアリング価格、決済方法を決定し、完全な市場サイクルの閉ループを完了します。

### 2. **EWA（経験加重吸引力）学習アルゴリズム**：

   従来のゲーム理論における政府の「完全合理性」という強い仮定を突破するため、本エンジンでは行動ゲーム理論分野の経験加重吸引力（EWA）学習アルゴリズムを導入し、現実の政府の限定合理性、適応的学習、政策進化の経路依存性の特徴を模倣しています。
   このアルゴリズムフレームワークでは、政府の各政策手段（例：法人税優遇の強度、消費割合）の複数の選択可能な強度値がそれぞれ「戦略」とみなされます。各戦略には動的に更新される「吸引力」値があります。各シミュレーション期間終了後、政府は現在の政策組み合わせがもたらす社会的厚生（炭素削減効果、グリーン電力取引量の伸び、政策財政コストを含む）を計算します。EWAアルゴリズムの中核は、実施された戦略の実際の社会的厚生に基づいてその吸引力を更新するだけでなく、「想像力」パラメータを通じて、選択されなかった戦略がもたらした可能性のある仮想の利益を加重更新することにあり、政策立案者の「反事実的思考」を模倣しています。
   吸引力値は特定のルール（減価率、学習強度などのパラメータを含む）に従って更新された後、政府が次期にある政策強度を選択する確率は、その吸引力の指数関数によって決定されます。これは「優位な戦略がより高い確率で選択されるが、探索の可能性も残る」という確率的応答モードを形成します。このプロセスにより、政府の政策調整は、漸進的、歴史的経験に基づくフィードバック、そしてある程度のランダムな探索性を伴う適応的学習プロセスとして表れ、それによって政策調整の経路と異なるシナリオ下での差別化された政策組み合わせを内生的に進化させることができます。

## 3. クイックスタート

### 3.1 環境準備
```bash
pip install -r requirements.txt
```

### 3.2 シミュレーション実行
すべてのデフォルトシナリオ（S0-S6）をカバーするバッチシミュレーションを直接実行できます。

```bash
# デフォルト実行（50年）
python run_simulation.py

# カスタム期間（例：20年）
python run_simulation.py --years 20
```

### 3.3 データ可視化
多次元分析チャートを生成します：

```bash
python visualize.py
```

## 5. パラメータ変更ガイド
### 中核説明
- プロジェクト特性: 「**依存関係ゼロ**」を実現しており、すべての初期状態は`core/base.py`で定義されています。
- 変更箇所: 調整が必要な唯一の設定ファイルは`core/base.py`です。

### パラメータグループ詳細
機能別に分類されたパラメータグループと主要な説明は以下の通りです：

1. **政府選好パラメータグループ**
   - 主要定数: **GOV_ALPHA**、**GOV_BETA**、**GOV_GAMMA**
   - 説明: 社会的厚生関数における**環境、経済、政策コスト**の意思決定ウェイト（フィッティング値: **4.44, 18.98, 5.2**）。

2. **企業設定パラメータグループ**
   - 主要定数: **GREEN_ENERGY_GROUPS**
   - 説明: クラスター分析に基づいて区分された**低規模グループ**（中小企業）と**中規模グループ**（大手企業）の初期資産、コスト、設備容量の設定。

3. **ユーザー設定パラメータグループ**
   - 主要定数: **CONSUMER_GROUPS**
   - 説明: 自然的独占属性に基づいて設定された**地域電力消費事業者**、**GDP弾力性係数**と**ARIMAモデル**を組み合わせて外生的に電力需要を設定。

4. **実験シナリオパラメータグループ**
   - 主要定数: **SCENARIO_TARGETS**
   - 説明: **S0-S6の7大シナリオ**（ベースライン、強力な供給側インセンティブ、強力な需要側牽引、対称的連携、地域差別化連携など）をカバー。

5. **学習アルゴリズムパラメータグループ**
   - 主要定数: **EWA_PARAMETERS**
   - 説明: 政府の適応的学習の中核パラメータ（**減価率、想像力パラメータ**などを含む）。


## 6. 出力解釈
シミュレーション結果は`batch_results/[タイムスタンプ]/`ディレクトリに統一して保存され、中核出力は以下の3種類に分類されます：

### 1. 総合結果表（**comprehensive_results.xlsx**）
**炭素削減量、グリーン電力浸透率、グリーン電力累計取引量、クリアリング価格、設備稼働率、累計政策コスト**などの多次元評価指標に関する詳細な時系列データを含みます。

### 2. サマリーレポート（**summary_report.txt**）
- 各政策シナリオの**総合パフォーマンス評価**と**ランキング**；
- **最適な政策連携経路**の推奨提案。

### 3. 可視化フォルダ（**visualizations/**）
4種類の主要なチャートを含みます：
- 主要指標の**長期進化トレンド折れ線グラフ**；
- 複数政策シナリオの**パフォーマンス比較レーダーチャート**；
- エージェント間相互作用の**ネットワークコード図**；
- 政策ツールの**動的呼び出しヒートマップ**。


## 7. 適用範囲
プロジェクトは4つの中核シナリオに焦点を当て、グリーン電力市場政策設計と進化分析をサポートします：

### 1. 政策シナジー効果評価
本シミュレーションエンジンの中核的応用は、異なる政策組み合わせのシナジー効果を体系的に評価することにあります。S0（ベースライン）からS6（地域差別化）までの7種類のシナリオを事前設定することで、エンジンは「強力な供給側インセンティブ」、「強力な需要側牽引」、および「対称的連携」などの政策組み合わせの長期的効果を定量的に比較できます。分析は、これらの組み合わせが、クリアリング価格と取引量の形成を含むグリーン電力市場の均衡状態にどのように影響するか、グリーン電力事業者の利益、設備稼働率、電力消費事業者のコストの変化に反映される経済運営にどのように作用するか、そして最終的に環境次元でいかに炭素削減を実現するかに焦点を当てます。シミュレーション結果は、なぜ「対称的連携」シナリオ（S3）が一側面の政策よりも経済効率向上と環境排出削減の「二重の配当」をより良く実現できるのかを直観的に明らかにし、政策立案者に定性的判断を超える定量的証拠を提供できます。

### 2. 市場主体戦略推論
エンジンは市場主体の異質性とその戦略的相互作用を精緻に描写します。供給側では、グリーン電力事業者は「中規模グループ」（大手企業）と「低規模グループ」（中小企業）に区分され、異なる初期資産、コスト構造、リスク選好を持っているため、同じ政策シグナルに直面しても、差別化された投資と価格設定の意思決定を行います。需要側では、電力消費事業者は地域のGDP成長と強制消費割当（RPS）という二重の制約を受けます。3段階スタッケルベルグ・ゲーム・フレームワークを通じて、本ツールはこれらの異質な主体が、政策圧力、割当目標、電力コスト変動の下で、短期的なコスト最小化の対応から、長期的な戦略的投資調整へとどのように進化するかをシミュレートし、それによってミクロ主体の行動がどのように集約され、マクロ市場のダイナミクスを形成するかを推論できます。

### 3. 長期進化経路の探究
モデルは、複雑システムの長期的ダイナミクスを観察するために特別に設計されています。50のシミュレーション期間（模擬年）にわたるスケールで、エンジンは主要指標の非線形軌跡を追跡できます。例えば、グリーン電力のクリアリング価格が初期の分化状態から、発電コストと代替価値によって決定される長期的均衡点へと徐々に収束する様子、投資の遅れ、政策学習、需要の不確実性のためにグリーン電力浸透率が持続的に非周期的な変動を示す様子、累積政策コストが自己強化的な正のフィードバック・メカニズムの下で特定の成長経路を形成する様子などを観察できます。これらの分析は、静的な均衡分析を超え、グリーン電力市場に内在する経路依存性、ロックイン効果、そして複雑適応システムとしての進化的特徴を明らかにすることを目的としています。

### 4. 地域差別化設計
エンジンは、「全国一律」と「地元の状況に適応」という2つの政策モードの効果テストをサポートします。「地域差別化連携」シナリオ（S6）を通じて、ユーザーは中国の「西電東送」の典型的な構造をシミュレートできます：西部の再生可能エネルギー資源豊富地域では、政策パラメータを供給側インセンティブ（例：より高い設備投資補助金）に重点を置くように設定し、発電投資を促進することができます。東部の負荷センター地域では、需要側の牽引（例：より高い消費割当と消費補助金）に重点を置き、グリーン電力の消費を確保し、地元ユーザーをインセンティブ付与することができます。このような差別化設計と全国統一政策（例：S3）の全体効率、地域間の公平性、地域間取引フローの面でのパフォーマンスを比較することで、より正確で適応性のある「一地域一政策」を策定するためのシミュレーション実験に基づく根拠を提供できます。
